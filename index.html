Cloudflare Pages
Cloudflare Pages 是由 Cloudflare(以下简称CF) 提供的静态网站部署服务
其允许用户上传网页文件，免费将网页部署至全球的服务器上
并且免费为网站加速，同时防止恶意攻击，例如DDoS

本网站的部署运用了 CF Pages ，如果本系统想要真正运用，大众化
建议购买域名，或自行搭建服务器

其他

{ } JSONBin
JSONBin是由Cloudflare进行加速/保护的数据库储存网站
该网站免费为用户免费提供每日10000条请求的额度
数据的下载与上载都由fetch()完成

本网站的数据储存同时使用了JSONBin与本地缓存
每日10000条请求额度对于这个项目来说足够了
同时为了提升数据的上载速度，将储存方式设为多个Bin储存
当然这样做，会导致连接与下载速度变慢
总之，是利大于弊的

如果你实在想提升连接速度与稳定性，建议使用Cloudflare Warp

其他

MathJax
MathJax是一个可以在网页中显示专业格式公式的库(或许你可以理解成插件)
它可以编译LaTeX为公式，例如下列这段LaTeX
x=\frac{-b\pm\sqrt{b^{2}-4ac}}{2a}
这将会被编译为 $x=\frac{-b\pm\sqrt{b^{2}-4ac}}{2a}$

这一功能完全免费，为网页插入公式提供了不少便捷
